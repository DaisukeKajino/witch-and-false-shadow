# vueify-boilerplate

## Overview

app: vueify + browserify + bower + debowerify

test: karma + mocha + chai + sinon + faker + factory_girl (on chrome)

## Install commands

```
$ sudo npm install -g gulp
$ sudo npm install -g bower
$ sudo npm install -g karma-cli
$ sudo npm install
$ bower install
```

# Build commands

```
$ gulp build
$ gulp build-test
$ gulp watch
```

# Test commands

```
$ karma start
```
